Debug null and log

    if (aa != null) {
        xxx;
    }

    这样的代码，今天遇到了 aa == null的情况，中间的代码没有执行。
    对null的情况打一个log利于调试，发现问题：
    if (aa != null) {
        xxx;
    } else {
        Log.e(TAG, "Error null aa");
    }
    
    如果 xxx; 代码太长，则这样更好一些:

    if (aa == null) {
        Log.e(TAG, "Error null aa");
    }
    xxx;
