How to share and rate you app
    Share app:

    1. Write a brief introduction and download address in string .
    2. Share the string with the below code.
    
    

    String content = getString(R.string.str_share_app_text);
    
    Intent it = new Intent(Intent.ACTION_SEND);
    it.putExtra(Intent.EXTRA_TEXT, content);
    it.setType("text/plain");
    startActivity(it);
    

    
    
    Rate app:

    Here just show the code to let user go to the main page of you app in app market.
    
    Replage "com.example.app" with you app's package name.
    
    

    Intent intent = new Intent(Intent.ACTION_VIEW); 
	intent.setData(Uri.parse("market://details?id=com.example.app")); 
	try {
        startActivity(intent);
    } catch (Exception e) {
        Toast.makeText(MemoSettingsActivity.this,
            R.string.str_error_no_proper_market, Toast.LENGTH_SHORT).show();
    }
